Gamepedia 用户手册

Gamepedia 是一款多平台游戏库聚合管理工具，旨在帮助您集中查看和管理来自 Steam、Epic Games、GOG 以及方块游戏平台（Cubejoy）的游戏。您无需切换多个客户端，即可在一个简洁的 Web 界面中浏览所有游戏、跳转商店页面、一键启动，并享受本地缓存的封面图与无限滚动体验。
功能特性

    🔐 统一授权管理 - 集中配置各平台的 API 密钥或授权码，一次登录，长久使用。

    🕹️ 多平台聚合 - 支持 Steam（含家庭共享）、Epic Games、GOG、方块游戏平台，更多平台持续开发中。

    🖼️ 智能图片缓存 - 游戏封面图首次访问时自动下载到本地，再次加载瞬间完成，支持离线浏览（需已缓存）。

    🔄 无限滚动与计数 - 浏览游戏列表时无需翻页，滚动即加载，右下角实时显示已加载数量。

    🚀 一键启动 - 点击游戏卡片上的“运行”按钮，即可通过各平台官方协议启动游戏（需客户端已安装）。

    🏠 Steam 家庭库标记 - 家庭共享的游戏卡片右上角显示“🏠 家庭库”角标，清晰区分。

    🔍 游戏搜索 - 快速过滤游戏名称，精准定位。

    🎨 自定义背景 - 支持上传个人图片作为背景墙纸（平铺模式）。

    🌐 国际化 - 自动根据浏览器语言切换中/英文界面。

安装与运行
前提条件

    Python 3.8 或更高版本

    现代浏览器（Chrome / Firefox / Edge）

    （可选）Tampermonkey 或 Violentmonkey 浏览器扩展（用于 GOG 和方块游戏的同步脚本）

下载与启动

    ⚠️ Windows 用户必看
    双击运行 setup.bat 即可一键安装，脚本会自动配置 Python 环境并安装所有依赖。

    ⚠️ Linux/Mac 用户必看
    双击运行 setup.sh 即可一键安装，脚本会自动配置 Python 环境并安装所有依赖。

    克隆或下载本项目代码到本地：
    bash

    git clone https://github.com/yourusername/gamepedia.git
    cd gamepedia

    安装依赖：
    bash

    pip install -r requirements.txt

    若没有 requirements.txt，请手动安装：
    bash

    pip install flask flask-cors requests

    启动程序：
    bash

    python app.py

    浏览器自动打开 http://localhost:5000，开始使用。

    首次启动后，请先进入“授权管理”完成各平台的授权（见下文）。

平台授权与同步指南

点击侧边栏的 “🔑 授权管理” 按钮，打开集中管理面板，按以下步骤配置每个平台。
平台	授权方式	所需凭证	同步操作
Steam	Web API Key + OpenID 登录	1. 申请 Steam Web API Key（填写 localhost 为域名）
2. 在授权管理中填入 API Key，点击“登录”完成 OpenID 授权	点击“同步”拉取所有已购买游戏
Steam 家庭组	油猴脚本	安装 family_sync.user.js 脚本，然后访问 Steam 商店 点击右下角按钮	自动同步家庭共享游戏
Epic Games	OAuth 授权码	1. 点击“获取授权码”跳转至 legendary.gl 页面
2. 登录 Epic 账号，复制授权码
3. 返回授权管理粘贴并提交	点击“同步”拉取游戏库
GOG	油猴脚本	安装 gog_sync.user.js 脚本，然后访问 GOG 账户页面 点击右下角按钮	自动抓取所有 GOG 游戏
方块游戏 (Cubejoy)	油猴脚本	安装 cubejoy_sync.user.js 脚本，然后访问 我的游戏 点击右下角按钮	自动分页抓取全部游戏

    注意：油猴脚本安装后，请在对应平台页面保持登录状态，并点击生成的“同步到游戏藏经阁”按钮。

界面操作说明

    侧边栏：显示所有已同步的平台标签，点击切换游戏列表。

    搜索框：输入游戏名称实时过滤当前平台游戏。

    游戏卡片：点击封面图跳转至对应商店页面（GOG 为搜索页面）；点击“运行”按钮启动游戏。

    右下角计数器：显示当前平台已加载游戏数 / 总数。

    回到顶部：滚动超过 300px 后右下角出现“↑”按钮，点击平滑返回顶部。

    设置：侧边栏底部的“⚙️ 设置”可自定义背景图片。

常见问题
Q: 游戏封面图不显示或显示占位图？

A: 首次同步时图片需要从网络下载，请检查网络连接并耐心等待。若多次失败，可尝试手动刷新浏览器（Ctrl+F5）。部分游戏可能不提供封面图，则显示平台占位图。
Q: Steam 家庭库同步失败？

A: 确保已安装油猴脚本，并在 Steam 商店页面登录后点击同步按钮。若仍失败，尝试重新安装脚本或重启浏览器。
Q: Epic 授权码过期？

A: 授权码有效期很短（通常几分钟），请尽快提交。若过期，重新获取即可。
Q: GOG 或方块游戏同步后看不到游戏？

A: 同步成功后需刷新游戏藏经阁页面（F5），侧边栏才会出现对应平台标签。如果仍未出现，检查数据库文件（gog_games.db / cubejoy_games.db）是否有数据。
Q: 如何更换语言？

A: 程序自动根据浏览器语言切换（中/英），如需强制更改，请修改浏览器语言设置或编辑 static/script.js 中的 userLang 变量。
Q: 缓存图片损坏如何修复？

A: 程序已内置图片验证与自动修复机制，访问游戏列表时若发现损坏图片会自动重新下载。也可手动删除 cache/ 目录下的平台子文件夹，重启程序后重新加载。
